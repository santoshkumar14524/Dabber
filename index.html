Basic Web Development Tools

Now you know what web development is (in a nut shell ofcourse) and decided to learn, the big hurdle would be where to start? Having lot of resources out there on the internet, like youtube, paid online courses like Udemy, Coursera, etc, free online courses like freeCodeCamp.org, it is really easy to get confused in choosing the right one. But before choosing the course you need to know what you will be learning in web development, so that you can have a better insight in choosing the courses.

  • Web Development Tools
  • HTML
  • CSS
  • JavaScript
  • PHP
  • SQL

HTML - Hyper Text MarkUp Language defines the basic structure of the webpage. You might be thinking 'OH! I got to learn another Programming Language', hold on. It's not a programming language its a markup language. I am not going to talk about the differences here but markup language is way much easier than any programmin language. To make it simple for you, it contains sort of templates for making the elements like paragraphs, links, images, headings, input fields etc.

CSS - Cascading Style Sheet provides the styling for the raw elements on the web page. CSS can be used to perform not only basic styling but also can be used to make beautiful animations ( even though animations can become way simpler using JavaScript ) and responsive webpages. There are tools like Bootstrap, Material CSS which provides mostly all the features (styling of elements and responsiveness to page) you want to use on your website.

JavaScript is used to provide interaction of the HTML elements and many more. You can create custom actions based on the user actions like what happens to the elements / element when user tries to interacts with them. JavaScript is not only used on the browser side ( I mean using JavaScript for modifying the HTML element on the browser / user side) but also on the server side using tools like NodeJs etc. There are also library for JS ( shortcut for JavaScript ) like JQuery.

PHP - Hypertext Preprocessor a recursive acronym. PHP acts like a bridge between the user side and server side, transmitting , receiving, data from database servers to browser and vice versa. All the form validation ( verfying the details email verification ) is mainly carried out with php. PHP also got some frame works just like JavaScript, those are Laravel, CodeIgniter.

SQL - Structured Query Language is designed for managing data in a relational database management system (RDBMS) or simply it is used for accessing and manipulating the data in databases. It is used along with tools like MySQL, PostgreSQL, SQLite to sumbit, change, retrieve data from databases.

Web Developer

Front End Developer is one who develops the front end side or the user side of the website. Like Creating and styling the HTML elements using CSS and using JavaScript to make user experience more easy and smooth. Got to be in touch with latest technology, frameworks. Deals with HTML, CSS, JavaScript and their frameworks, libraries.

Back End Developer is one who develops the back end side or the server side of the website. Receiving data from users through front end and passing them to servers and sending the response to browser, where the front end developer styles the response. Deals with PHP, MySQl, SQL and their frameworks and libraries.

Full Stack Developer is one who have the knowledge of both front end and back end developers. So now you know why there is a more demand for full stack developers than front and back end developers alone.
